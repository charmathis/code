🌧 Cloudburst Monitoring System

This system monitors extreme rainfall conditions across Tamil Nadu districts using live sensor data stored in Firebase. The dashboard displays rainfall intensity, humidity levels, and risk status in real-time.

When rainfall exceeds safe thresholds, the system automatically generates warnings and danger alerts to indicate possible flood situations. The map view highlights districts with colored indicators for quick monitoring.

This project helps in early disaster detection and improves preparedness against sudden cloudburst events and floods.
